# product-store-app-using-MERN
# 🛒 Product Store App

A full-stack MERN application that allows users to create, view, update, and delete products with a clean UI built using Chakra UI and state management powered by Zustand.

---

## 🚀 Features

- Add new products with image, name, and price
- Update and delete products
- Responsive UI with Chakra UI
- Light/Dark mode toggle
- Toast notifications for actions
- Backend API using Express & MongoDB
- Vite-powered React frontend with Zustand

---

## 🧰 Tech Stack

**Frontend:**
- React
- Vite
- Zustand
- Chakra UI
- React Router

**Backend:**
- Node.js
- Express.js
- MongoDB with Mongoose
